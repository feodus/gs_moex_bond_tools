# Политика безопасности

## Обзор
Этот документ описывает политику безопасности для проекта MOEX Bond Data for Google Sheets. Мы стремимся обеспечить безопасность использования нашего кода и API.

## Поддерживаемые версии
Поскольку это библиотека Google Apps Script, которая работает в песочнице Google, поддержка версий осуществляется для последней активной версии проекта.

## Сообщение о уязвимостях
Если вы обнаружите уязвимость в безопасности, пожалуйста, сообщите о ней, используя один из следующих способов:

1. Создайте issue в репозитории с меткой `security` (если это возможно)
2. Отправьте сообщение через систему контактов GitHub

Пожалуйста, предоставьте как можно больше информации о найденной уязвимости, включая:
- Описание уязвимости
- Шаги для воспроизведения
- Потенциальное воздействие
- Возможные решения

## Обработка уязвимостей
Мы стремимся ответить на сообщения о безопасности в течение 48 часов. После получения уведомления мы:
1. Подтвердим получение сообщения
2. Оценим уязвимость, включая её диапазон и серьёзность
3. Определим решение
4. Выпустим исправление
5. Опубликуем объявление о безопасности

## Рекомендации по безопасности
При использовании этой библиотеки рекомендуется:
- Не хранить чувствительные данные в открытом виде в Google Таблицах
- Проверять все данные, полученные из внешних API
- Обновлять библиотеку до последней версии
- Ограничивать права доступа к таблицам, использующим эту библиотеку

## Дополнительная информация
Эта библиотека использует API Московской Биржи и работает в рамках ограничений Google Apps Script. Все сетевые запросы осуществляются через встроенные службы Google, что добавляет уровень безопасности.
