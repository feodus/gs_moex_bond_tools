Создай презентацию на Gamma.app для портфолио, демонстрирующую навыки разработки Google Apps Script. Тема: "MOEX Bond Data for Google Sheets" - библиотека для получения данных по облигациям с Московской Биржи.

Слайд 1: Заголовок
- Название: "MOEX Bond Data for Google Sheets"
- Подзаголовок: "Интеграция финансовых данных MOEX в Google Таблицы"
- Описание: Библиотека Google Apps Script для получения данных по облигациям с Московской Биржи в реальном времени. Позволяет использовать данные об облигациях (цены, даты купонов, размеры купонов, даты погашения) прямо в ячейках Google Таблиц через кастомные функции.

Слайд 2: Функциональность
- Заголовок: "Основные возможности"
- Текст:
  - GET_MOEX_PRICE - текущая цена облигации
  - GET_MOEX_NAME - наименование облигации
  - GET_NEXT_COUPON - дата следующего купона
  - GET_COUPON_VALUE - размер следующего купона
  - GET_MATURITY_DATE - дата погашения облигации
  - GET_NEAREST_OPTION_DATE - ближайшая дата опциона (put/call) или амортизации
  - Поддержка флоатеров (облигаций с плавающей ставкой)
  - Система кэширования для оптимизации запросов

Слайд 3: Технические особенности
- Заголовок: "Техническая реализация"
- Текст:
  - Google Apps Script с runtime V8
  - Интеграция с MOEX ISS API
  - Интеллектуальное кэширование (цены: 5 мин, даты: 6 ч, наименования: 24 ч)
  - Защита от превышения лимитов API
  - Обработка ошибок и fallback-механизмы
  - Массовое обновление данных с задержками
  - Открытый исходный код (MIT License)
