# История изменений

Все заметные изменения в этом проекте будут задокументированы в этом файле.

## [Не выпущено]

### Добавлено

- Основные функции для получения данных по облигациям MOEX:
  - `GET_MOEX_PRICE()` - получение текущей цены облигации
  - `GET_MOEX_NAME()` - получение наименования облигации
  - `GET_NEXT_COUPON()` - получение даты следующего купона
  - `GET_COUPON_VALUE()` - получение размера следующего купона
  - `GET_MATURITY_DATE()` - получение даты погашения
  - `GET_NEAREST_OPTION_DATE()` - получение ближайшей даты опциона (put/call) или амортизации
- Система кэширования для оптимизации запросов к API
- Меню "MOEX" в Google Таблицах для массового обновления данных
- Защита от превышения лимитов API с помощью задержек между запросами
- Поддержка флоатеров (плавающих ставок) в расчетах купонов

### Изменено

- Обновлены методы получения данных для большей надежности
- Улучшена обработка ошибок и исключений

### Исправлено

- Исправлены потенциальные проблемы с парсингом данных
- Улучшена стабильность работы с API MOEX
