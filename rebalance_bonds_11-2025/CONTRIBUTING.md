# Как внести свой вклад (Contributing)

Мы рады любым вкладам в развитие проекта! Если вы нашли ошибку или хотите предложить улучшение, пожалуйста, следуйте этим инструкциям.

## Сообщение об ошибках

Если вы столкнулись с ошибкой:

1. Проверьте вкладку [Issues](issues), возможно, ошибка уже известна.
2. Если нет, создайте новый Issue, подробно описав проблему, шаги для воспроизведения и ожидаемое поведение.

## Предложение изменений (Pull Requests)

1. Форкните (Fork) этот репозиторий.
2. Создайте новую ветку для ваших изменений (`git checkout -b feature/my-feature`).
3. Внесите изменения и закоммитьте их (`git commit -m 'Add some feature'`).
4. Запушьте ветку в свой форк (`git push origin feature/my-feature`).
5. Откройте Pull Request в этот репозиторий.

## Стиль кода

Пожалуйста, придерживайтесь существующего стиля кода (используйте `const`, понятные имена переменных, JSDoc комментарии).
